# Notion-Inspired Planner Sidebar Notes

Sources reviewed:

- Notion help: [Navigation with the sidebar](https://www.notion.com/de/help/navigate-with-the-sidebar)
- Notion help: [Pages and blocks](https://www.notion.com/de/help/category/write-edit-and-customize)

Applied principles:

- Treat the left panel as an app rail, not a content card.
- Prefer compact named sections with quiet hover rows.
- Keep primary actions close to the workspace title.
- Use small visual keys for layers and block types instead of large explanatory cards.
- Put workspace metadata and settings links in a footer-like area.
- Avoid copying Notion UI directly; use neutral density, lightweight borders, and block-oriented structure as inspiration.
