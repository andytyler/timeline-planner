# Interaction Motion

Use the combined buffer handle as the planner motion standard.

- Resting controls should be compact, low contrast, and visibly actionable without extra boxes.
- Hover and focus should expand the target, deepen contrast, and reveal direction only when direction matters.
- Drag creation should show the concrete block size and position while the pointer moves.
- Keep motion short: 150-200ms, ease-out, using size, opacity, color, shadow, and transform.
- Avoid decorative animation, nested cards, and instructional chrome inside planner blocks.
